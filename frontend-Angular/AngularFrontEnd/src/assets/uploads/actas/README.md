# Directorio de Actas

Este directorio almacena las actas de comité subidas por las secretarias académicas al aprobar eventos.

Los archivos se nombran con timestamp para evitar colisiones:
- Formato: `{timestamp}_{nombre_original}.pdf`
- Ejemplo: `1700000000000_acta_comite_nov_2025.pdf`

**Nota:** Este directorio debe tener permisos de escritura para el servidor backend.
